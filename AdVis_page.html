Interactive Ray-casting volume rendering & Preintegrated isosurface visualization

Demo video can be found HERE.

The project is about visualiza volume data using different approaches.

Raycasting: Imagine there is a ray emitting from each pixel in the frame. Find the intersections of ray and volume as entry and exit points. Composite along the ray every sample distance with corrected voxel color and transparency in this ray segment.

Preintegration: Composite all the voxel values within the two voxels sampled in Raycasting, so that the transition of opacity between the two sample point would not be missed.

Lighting: Implement Blinn Phong Lighting model.
